Computer Graphics

Washington University in St. Louis, Spring 2013

Labs were written in C++ using provided base-code. Concepts covered include:
  • The scan line algorithm
  • 3D shape tessellation
  • Camera adjustments (clipping planes, aspect ratio, projection matrices)
  • Ray-shape intersection
  • Converting a hierarchical scene description to its visual representation
  • Rendering using ray tracing (phong lighting, shadows, attenuation, reflection)

The first labs served as an introduction to various concepts in computer graphics, and the project culminated with creating your own scene environment and rendering it using ray tracing.
