Artificial Intelligence

Washington University in St. Louis, Spring 2012

This course is detailed on the Wash U website here. I worked in a team of two people on all of the projects (except project 0 which was a Python tutorial).

There were four project topics: state-space search, multi-agent search, probabilistic inference, and reinforcement learning. In each project, we implemented general-purpose AI algorithms in Python and then integrated them into the Pac-Man world using search heuristics, evaluation functions, and feature functions, in order to achieve goals in the game, such as collecting all the food and avoiding ghosts.

For the 1st project, we implemented depth-first search, breadth-first search, A* search and uniform-cost graph search, as well as heuristics to find all the corners of the maze and eat all the dots in the maze.


For project 2, we implemented intelligent agents for the game, both for Pac-Man and the ghosts. For this project, we wrote agents that used minimax, alpha-beta pruning, expectimax, as well as designed evaluation functions for the agents.


In project 3, we explored Markov decision processes and reinforcement learning. The Q-learning agent learns by trial and error from interactions with the environment, so pacman learns when to go to eat the food pellets and when to run from ghosts.


Project 4 switched the tables and turned Pac-Man into a ghost hunter! Given that Pac-Man couldn't see the ghosts, but could only track them using sensors, our goal was to write a program that enables Pac-Man to hunt and eat ghosts. The topics we dealt with were Belief Networks and particle filters (approximate inference).


Finally, at the end of the course we had a Pac-Man capture-the-flag (or the food pellets) competition in which the Pac-Man and the ghosts on one side of the maze formed a team that was supposed to capture the food on the other side of the maze while protecting the food on their side. The game pitted two teams against each other.
