Projects > Wash U Courses > Computer Graphics

Washington University in St. Louis, Spring 2013

Labs were written in C++ using provided base-code. The first labs served as an introduction to various concepts in computer graphics. The concepts covered included:
  • The scan line algorithm
  • 3D shape tessellation
  • Camera adjustments (clipping planes, aspect ratio, projection matrices)
  • Ray-shape intersection
  • Converting a hierarchical scene description to its visual representation
  • Rendering using ray tracing (phong lighting, shadows, attenuation, reflection)















The project culminated with creating your own 3D scene environment and rendering it using ray tracing. You can see my scene above. On the left is a basic rendering and on the right is a rendering done using the ray tracing code I wrote (smaller because ray tracing is costly and slow).
