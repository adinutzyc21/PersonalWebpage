Projects > Wash U Courses > Advanced Machine Learning

Washington University in St. Louis, Fall 2012

This course had 3 labs that involved implementing machine learning papers in Matlab, but unfortunately I do not have that code anymore.

The final project for the class, Middle Cerebral Artery Aneurysm Classification, was a team project to automate the pre-operatory aneurysm classification process. We used shape analysis techniques paired with machine learning over a small set of manually categorized aneurysm meshes, as shown in the figure below. The machine learning concept we used was random forests, which were particularly suited for our set of data. We wrote the code in C++ for MATLAB via mex files, as well as compared to a native Matlab implementation.


For the preliminary results, we obtained an overall classification accuracy of 0.7, while doing better on anneurisms for which we had better data and features. One of our team members continued the project and obtained better results with more data and a better training dataset.

Special thanks to Sam and Derek for working with me on this!
