Projects > Wash U Courses > Artificial Intelligence

Washington University in St. Louis, Spring 2012

Implemented general-purpose AI algorithms in Python and then integrated them into the Pac-Man world using search heuristics, evaluation functions, and feature functions, in order to achieve goals in the game, such as collecting all the food and avoiding ghosts.
The course topics were: state-space search, multi-agent search, probabilistic inference, and reinforcement learning.

Project 1


Implemented depth-first search, breadth-first search, A* search and uniform-cost graph search, as well as heuristics to find all the corners of the maze and eat all the dots in the maze.

Project 2


Implemented intelligent agents for the game, both for Pac-Man and the ghosts.
For this project, we wrote agents that used minimax, alpha-beta pruning, expectimax, as well as designed evaluation functions for the agents.

Project 3

Explored Markov decision processes and reinforcement learning.
The Q-learning agent learns by trial and error from interactions with the environment, so Pac-Man learns when to go to eat the food pellets and when to run from ghosts.

Project 4


The tables were switched and Pac-Man was a ghost hunter! The goal was to write a program that enables Pac-Man to hunt and eat ghosts, given that he couldn't see them, but could only track them using sensors.
The topics we dealt with were Belief Networks and particle filters (approximate inference).

Final Project

Pac-Man capture-the-flag (or the food pellets) competition in which Pac-Man and the ghosts on one side of the maze formed a team that was supposed to capture the food on the other side of the maze while protecting the food on their side. The game pitted two teams against each other.


This course is detailed on Wash U's website.
